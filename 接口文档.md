## 插入用户/更新用户

方法：POST

URL：http://localhost:9090/user

**用法1 传入其他参数，但是不带id**

实例：

传入：

```JSON
{
        "name": "新插入对象",
        "nickname": "大鹏元帅",
        "password": "123456",
        "avatar_url": "https://m.duitang.com/blog/?id=1391351457&belong_album=109495738",
        "sex": "男",
        "email": "89999999@qq.com",
        "phone": "+867894561",
        "address": "天界"
}
```

返回格式

```
1
```

表示插入成功

## 登录 （简陋）

方法：GET

URL：http://localhost:9090/user/login

参数 JSON

```json
{
    "name":"白骨精",
    "password":"123456"
}
```

返回这个用户的全部信息

```json
{
    "id": 4,
    "name": "白骨精",
    "nickname": "白骨夫人",
    "password": "123456",
    "avatar_url": null,
    "sex": "男",
    "email": "89999999@qq.com",
    "created_at": "2022-05-31 11:20:08",
    "updated_at": null,
    "phone": "+867894561",
    "address": null
}
```



## 删除用户

方法：DELETE

URL：http://localhost:9090/user/delete/{id}

例：删除id为7的用户

发送DELETE http://localhost:9090/user/delete/7

用户即被删除 返回1
