# 消息接口

## Get方法 

messages

1.直接分页展示全部数据

URL :  http://localhost:9090/M/page?pageNum=1&pageSize=2

2.按照群组分页产生最新消息展示

URL : http://localhost:9090/M/page/s?pageNum=1&pageSize=2



传入：pageNum 第几条数据开始，从1开始 

pageSize 页面展示数据量



返回格式：
